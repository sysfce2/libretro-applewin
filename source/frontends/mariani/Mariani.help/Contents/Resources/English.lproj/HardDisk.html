Configuring a Hard Disk

Mariani supports the emulation of hard disk images for higher speed and capacity storage.

To set up an existing hard disk image

  1. Open Preferences and navigate to the Computer pane.
  2. Select Generic Hard Disk Drive for slot 7.
  3. Navigate to the Storage pane.
  4. Click on the Hard Drive option, and select the appropriate .hdv disk image.
  5. Reboot the emulator (such as by closing the Preferences window.)

To create a new hard disk image

  1. Open Preferences and navigate to the Computer pane.
  2. Select Generic Hard Disk Drive for slot 7.
  3. Navigate to the Storage pane.
  4. Click on the Create New Hard Disk button.
  5. Enter a name and select a location for the .hdv image, then click Save.
  6. Insert a ProDOS system disk image into slot 6, drive 1.
  7. Reboot the emulator while holding down the    key. The computer will boot with the floppy drive instead of the hard drive.
  8. Select System Utilities from the main menu.
  9. Select Copy Files, from Slot 6 Drive 1, to Slot 7 Drive 1.
  10. Select Some Files, and then follow instructions on-screen to select the system files you wish to copy to the hard disk. You will at least need to the PRODOS and BASIC.SYSTEM files to the hard disk.
  11. Reboot the emulator again to boot from the hard disk.

You will see an additional circle button representing the newly-added hard disk.

Reconfiguring the hard disk while it is in operation may result in a corrupted disk image, analogous to shutting off the real computer or unplugging the hard disk card from its slot.
